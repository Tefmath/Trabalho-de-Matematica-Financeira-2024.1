# Trabalho-de-Matematica-Financeira-2024.1
Trabalho feito na disciplina de Matemática Financeira na UFRJ em 2024.1

Professor Rafael Souza dos Santos

Grupo: João Pedro Lima Bittencourt Silva, Rhana Gomes de Santana Santos, Stefane Moraes Castoldi

Neste trabalho explicamos como funcionam as instituições financeiras e apresentamos os principais produtos bancários no Brasil, destacando suas características, vantagens, desvantagens e riscos, especialmente relacionados aos juros e ao endividamento.
